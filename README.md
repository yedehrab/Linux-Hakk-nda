## Depolar

- `/etc/apt/sources.list/` Resmi depolar
- `/etc/apt/sources.list.d/` dizininde uygulama depoları saklanır.
- `thunar` komutu ile dosya gezginleri ile açabilirsin. (*thunar pardus içindir*)
